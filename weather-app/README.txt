# Weather Forecast App

This app shows 5-day weather forecasts for Brisbane, Gold Coast, and Sunshine Coast.

## How to Use

### Install:
1. `composer install`
2. `npm install`
3. Set your API key in `.env`

### Run:
- `php artisan serve` to start the web server
- `php artisan forecast Brisbane` to run the forecast in the terminal

## Notes
- Frontend is built in Vue.js
- Backend is Laravel
- Data comes from WeatherBit API
